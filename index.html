05 · projects, brick by brick

Everything is built from bricks

Favorite film: Whiplash. Favorite movie: The Lego Batman Movie. Yes, those are different categories. This shelf is still young and I keep snapping on new bricks.

SeatLive

real-time booking engine

Stops two people from booking the same seat, even under real concurrency: atomic conditional UPDATEs with Redis TTL holds, idempotency keys on confirmation, and live seat maps over WebSockets. CI fires 50 simultaneous requests at one seat and checks that exactly one wins. Load test: 4,459 requests, zero failures.

Python · FastAPI · PostgreSQL · Redis · React · Docker

Edge-ML audio benchmark

55× smaller, still listening

Took a 3.9 MB audio-classifier network and compressed it to 71 KB through distillation, INT8 quantization, and pruning, then ran it on a five-dollar microcontroller. About 300 ms per inference in roughly 300 KB of RAM, with a PTQ vs QAT comparison to back it up.

PyTorch · TFLite Micro · C++ · ESP32-S3

STM32 motor controller

bare metal, no training wheels

Firmware with every driver written from scratch, no HAL: GPIO, UART, SPI, I2C, timers. Closed-loop PID speed control with under 2% steady-state error, later ported onto FreeRTOS with a telemetry pipeline that plots step responses in Python.

Embedded C · STM32 · FreeRTOS · PID
