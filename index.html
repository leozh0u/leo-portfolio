🐉 LLOYD

BlunderNet Arena ↗

serving the engine at scale

A Go platform that puts BlunderNet online: stateless API fleet behind an ALB, inference decoupled onto an SQS-fed ONNX worker fleet that autoscales on queue depth. p99 move latency stays under 250 ms, and killing an API task mid-game is invisible to players. Load-tested to 4,000+ req/s with zero errors; the whole VPC/ECS/ALB/RDS/SQS stack is Terraform, running under $5/month.

Go · AWS · Terraform · Redis · ONNX · k6

❄ ZANE

BlunderNet ↗

self-training chess engine

AlphaZero-style engine — residual CNN with policy and value heads, PUCT Monte-Carlo tree search in C++ (1.8× faster via virtual-loss batching). Retrains itself daily on fresh Lichess games through scheduled GitHub Actions, tracked against a 1,200-puzzle tactics suite and daily Elo runs versus Stockfish.

PyTorch · C++ · pybind11 · GitHub Actions

⚡ JAY

SeatLive ↗

concurrency-safe booking engine

Stops double-booked seats under real concurrency — atomic conditional UPDATEs with Redis TTL holds, idempotency keys on confirmation, live seat maps over WebSockets. CI fires 50 simultaneous requests at one seat and proves exactly one wins; Locust served 4,459 requests with zero failures.

FastAPI · PostgreSQL · Redis · React · Docker · Sentry
